### Group 2 — 상단 마켓바 마퀴 (우→좌 흐름)
- **원인 발견**: 사용자의 OS "동작 줄이기"가 ON → `@media (prefers-reduced-motion: reduce)` 매칭으로 정지됨
- **해결**: 마퀴 한정 `prefers-reduced-motion` 예외 제거 (사용자 명시 요청 기능)
- **속도**: `data.length × 2.2초` (12개 → 26초)
- 무한 루프: 시퀀스 2배 복제 + `translateX(0% → -50%)`
- hover 시 일시정지

### Group 3 — SK하이닉스 AI 분석 429 진단 + 해결
- **원인**: Anthropic API rate limit (분당 토큰 한도). 연속 분석 시 발생.
- **해결** (`backend/main.py:904~`):
  - `_ai_error_message()` 도입 — 429/401/400/529/5xx 한국어 안내
  - `_anthropic_post()` 헬퍼 — 429/529 시 `Retry-After` 헤더 존중 자동 1회 재시도 (2~8초 백오프)

### Group 4 — 보유 탭 UX

### Group 5 — 비중 탭 시각화 강화
